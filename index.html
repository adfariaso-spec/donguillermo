🤖 Backup Automático Diario

El sistema genera automáticamente un backup completo cada 24 horas cuando alguien abre la app. Incluye todas las tablas y el código fuente, descargado a la carpeta Descargas con el prefijo Bkp_DG_AUTO_.

💡 Recomendación: mueve estos archivos a Google Drive o pendrive periódicamente y borra los antiguos para no llenar Descargas.

💾 Exportar Backup
💾

Descarga un archivo .json con todo el sistema:
productos, clientes, movimientos, ventas, usuarios,
comunas, proveedores, accesos y código fuente.
Guárdalo en Google Drive, OneDrive o un pendrive.

Se descargará como Bkp_DG_FECHA_HORA.json

📂 Importar Backup
📂

Restaura todos tus datos desde un backup anterior.
⚠️ Esto reemplazará todos los datos actuales.
Asegúrate de exportar primero si tienes datos nuevos.

Solo archivos .json generados por este sistema
